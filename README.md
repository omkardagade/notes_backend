# Notes Backend API

A REST API built using Node.js, Express.js, MongoDB Atlas, and Mongoose.

## Features

- Create Notes
- Get All Notes
- Get Single Note
- Update Notes
- Delete Notes

## Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

## Run Locally

```bash
npm install
npm start
```

Create a `.env` file:

```env
MONGO_URI=your_mongodb_connection_string
```

Server runs on:

```text
http://localhost:3000
```
